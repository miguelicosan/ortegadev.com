# ReviCar - Plataforma de Gestión Vehicular con IA

**La primera aplicación con inteligencia artificial para gestionar el mantenimiento de tu vehículo de forma predictiva.**

## Resumen Ejecutivo

ReviCar es una aplicación web y móvil nativa (iOS/Android) que revoluciona la gestión del mantenimiento vehicular mediante algoritmos de IA predictiva. Calcula ahorros automáticos, predice costes futuros y optimiza el cuidado de tu vehículo en tiempo real.

## Características Técnicas Destacadas

- **IA Predictiva**: Score dinámico (0-100) basado en cumplimiento, precisión IA, estado vehicular y optimización de costos
- **Base de Datos Vehicular**: 68 marcas, 576 modelos desde 1980 hasta 2025
- **Apps Nativas**: iOS y Android desde un mismo codebase con Capacitor.js
- **Autenticación Robusta**: OAuth social login (Google, Apple) con email verification
- **Responsive Mobile-First**: Optimizado para todos los dispositivos
- **CRUD Completo**: Gestión de vehículos, historial de mantenimiento, programación inteligente

## Stack Tecnológico

**Frontend**: React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui  
**Mobile**: Capacitor.js 7.4 (iOS/Android)  
**Backend**: Supabase (PostgreSQL) + Edge Functions  
**IA**: Algoritmos predictivos + OpenAI/Gemini  
**DevOps**: GitHub + Hostinger + SMTP
